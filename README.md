# 问题定位助手 · 用户手册

桌面工具：抓取本地 **Outlook 邮件** / **WeLink 群聊**，按规则匹配后归档为经验。
支持两种模式：**在线**（推送后端，做 OCR / LLM 整理 / 入库到团队知识库）与 **离线**（不连服务器，只在本地产出 `.html` + `.md`）。
Windows 单个 exe，双击即用，无需安装 Python。

---

## 一、安装与启动

1. 拿到 `extension.exe`，双击运行。
2. 首次启动会弹出「**初始配置**」，填完即可进入主界面。
3. 关闭窗口是**最小化到系统托盘**（定时同步继续在后台跑）；彻底退出请右键托盘图标 → 退出。

---

## 二、初始配置

| 项 | 说明 |
|---|---|
| **服务器** | 下拉选择：`云核心网`（团队后端）/ `离线（仅本地导出）`；也可选「手动输入 URL」填自建后端。 |
| **工号** | 输入姓名或工号搜索，从下拉结果里选（在线必填；离线可留空）。 |
| **Namespace** | 知识库分区（在线必填，点「刷新」拉取列表）。 |
| **文件保存目录** | 生成的 `.html` / `.md` 落盘位置（离线必填）。 |
| **额外配置** | 可选 JSON，随每封邮件作为扩展字段上报。 |

- **在线**：处理结果推送后端，做去重 / OCR / LLM 整理 / 入库，沉淀到团队知识库。
- **离线**：不连服务器，邮件处理只在本地生成 `.html` + `.md` 两份文件，不做 OCR / LLM / 入库。
- 弹窗里的「**用户手册及内源代码仓地址 ↗**」可点击跳转到源码仓与手册。

## 三、邮件

- **左侧「文件夹」**：勾选要纳入处理 / 定时范围的 Outlook 文件夹（勾选后即为扫描范围）。
- **筛选**：`全部` / `按规则匹配` 两个分段；右侧搜索框按主题 / 发件人过滤。
- **规则 / 黑名单**（点「规则」打开）：
  - 规则（白名单）按 **主题关键词 / 正文关键词 / 发件人** + **OR / AND** 逻辑匹配。
  - 最终命中集合 = **规则集合 − 黑名单集合**。
  - 规则本地存储，只增删改。
- **刷新邮件**：读取勾选文件夹的邮件列表，标出命中规则的邮件。
- **处理选中**：勾选若干邮件 → 在线推送后端处理 / 离线本地导出（勾选=强制处理，已解析的会重跑）。
- **启动定时**：可选「每隔 N 分钟」或「每天某时刻」，自动把**命中规则**的邮件同步一次。
- **状态列**（在线）：`已解析` / `解析中` / `解析失败`，来自后端解析队列。
- **底部日志**：默认折叠成一行，点「日志 ▸」展开查看运行明细。

## 四、WeLink

### 定位过程记录
监听指定群，靠**聊天命令**录制定位过程并归档为经验。

- **命令**（可在界面自定义）：`开始定位` / `结束定位` / `总结经验`。
- **总结命令**用法：`<总结命令> 姓名 工号 日期 时间`
  - 带 **1 个时间**：该时间点为**起点**，截止到总结命令本身。
  - 带 **2 组时间**：显式区间（顺序写反会自动纠正）。
- **按天自动记录**：勾选后每日某时刻自动归档全天记录。
- **监听的群聊**：填群组 ID + 名称添加。
- 进入页签**默认不自动开监听**，点「开始监听」才启动。

### 自动回复
关键词触发自动回复。

- **监听对象**：`群组` / `用户` 两类（分段切换）。
- **关键词规则**：命中关键词 → `AI 回复`（填 Prompt，走后端大模型）或 `固定回复`（直接发预设内容）。
- **置顶会话数**：WeLink 置顶会话排在最前，填入数量以跳过它们、读到真正最新的会话。

## 五、常见问题

- **文件夹列不出来**：先点文件夹面板的「刷新」；确认 Outlook 已启动并登录。
- **下拉箭头 / 复选框显示异常**：重启客户端。
- **定时同步没反应**：确认已「启动定时」，且窗口已最小化到托盘（后台运行）。
- **离线模式没生成文件**：确认「文件保存目录」已填且有写权限。
